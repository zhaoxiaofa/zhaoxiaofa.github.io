1. 原理

LinkedList 底层是双向链表，对应的是链表这种基本的数据结构。同样，它的大多数特性都来源于链表。

我们看看 LinkedList 类的继承结构：

如代码所示，LinkedList 实现了 Deque、Cloneable、Serializable。

  • Deque：Deque 是 Queue 的子接口，双向队列
  • Cloneable：支持克隆
  • Serializable：支持序列化

缺点：

不支持随机读，读取数据需要遍历。时间复杂度O(n) 。

优点：

  1. 非常适合频繁的插入
  2. 可以当做队列来使用

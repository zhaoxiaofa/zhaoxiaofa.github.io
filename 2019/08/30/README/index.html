引言

此网站所有文章均为我日常工作、学习总结的笔记，在写的过程中参考了相关的书籍、文章、源码，参考借鉴之处会尽量标出。当然，有些笔记由于时间久远，已忘记出处，望见谅！若涉及版权问题，请联系我。

文章特点

  1. 少说话、说人话：能用一句话说完的坚决不写一段话，尽量少说废话；我个人不喜欢艰涩的概念，所以文章都使用大白话，尽量做到通俗易懂。
  2. 画图：在做程序员之前我是一名建筑师，我深知，很多时候，一张图胜过二百字，所以文章中也会插入很多辅助理解的图。
  3. Test + Debug：涉及到源码的地方，我会写单元测试，一步步 debug，没有什么比直接运行看效果来得更直观了。同样的，自己多 debug 几次，也会加深理解和记忆。

致谢

由于本人能力有限，难免有错误和遗漏之处，恳请大家批评指正。

如果你觉得文章对你有帮助，可以移步到我的 GitHub 点个 star 。
